¿Kinsta Ofrece Integración Git?

Actualizado June 02, 2017

Tenemos disponible el cliente Git, así usted puede usarlo, o sea mandar a través de SSH (intérprete de órdenes seguro) y retirar de su repositorio Git de Github, Gitlab, Bitbucket o de cualquier lugar, pero aún no ofrecemos la posibilidad de integración Git automáticamente. Esta característica vendrá pronto.

Después del lanzamiento puede crear un script de implementación en unos minutos que puede mandar mediante SSH a su contenedor Google Cloud Linux y retirar de allí una vez que su repositorio remoto contenga la versión nueva. Así en vez de introducir “git push kinsta/my_site” debe añadir “git push kinsta/my_site && ssh my_site@1.2.3.4 && cd /www/my_site/public && git pull”.

Muchos de nuestros clientes utilizan WP Pusher también.

Si tiene alguna pregunta, nuestro departamento de soporte estará encantado de ayudar una vez que esté listo y ejecutando.
